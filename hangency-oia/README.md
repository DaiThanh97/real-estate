# Backend

### Start

```console
npm start
```

### Test

```console
npm t
```

### Format

```console
npm run format
```

### Migrate

```console
npm run typeorm:migrate
```

### Revert migration

```console
npm run typeorm:revert
```

### Generating migrations

```console
npm run typeorm:generate-migration --name=MIGRATION_NAME
```

### Creating migrations

```console
npm run typeorm:create-migration --name=MIGRATION_NAME
```
